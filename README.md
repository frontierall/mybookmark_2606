# mybookmark_2606

AI 서비스와 실무에 유용한 웹사이트를 카테고리별로 모아 보는 정적 북마크 허브입니다.

## 현재 구성

- 총 140개 사이트를 8개 상위 카테고리로 분류합니다.
- 상위 카테고리: AI 챗봇, 이미지/비디오, 음악/오디오, 개발 도구, 글쓰기/생산성, 트렌드/시장 인사이트, AI 플랫폼, 강의/학습
- `트렌드/시장 인사이트`에는 마케팅/소비자, 시장 데이터, IT/AI 뉴스, 디자인 레퍼런스 사이트를 모았습니다.
- 최근 추가 사이트에는 `addedAt` 날짜를 기록하고, 최근 30일 안에 추가된 항목만 `NEW` 배지를 표시합니다.

## 주요 기능

- 상단 카테고리 탭과 좌측 하위 카테고리 필터
- 이름, 설명, URL, 상위/하위 카테고리를 대상으로 하는 전역 검색
- 자주 쓰는 사이트 상단 고정
- 카드 좌상단 별 버튼으로 즐겨찾기 추가/해제
- 즐겨찾기 상태는 브라우저 `localStorage`에 저장됩니다.
- 라이트/다크 모드 전환
- 모바일에서는 사이드바를 숨기고 검색창과 필터 중심으로 탐색합니다.

## 데이터 관리 규칙

사이트 데이터는 현재 `index.html` 안의 `CATS`, `DB` 배열에서 관리합니다.

서비스 항목 기본 형식:

```js
{
  name: '서비스명',
  url: 'https://example.com/',
  i: 'E',
  c: '#2563eb',
  cat: 'insight',
  sub: 'technews',
  desc: '서비스 설명',
  addedAt: '2026-06-29',
  isFavorite: true
}
```

- `cat`은 `CATS`의 상위 카테고리 `id`와 일치해야 합니다.
- `sub`는 해당 상위 카테고리의 하위 카테고리 `id`와 일치해야 합니다.
- `addedAt`은 `YYYY-MM-DD` 형식으로 기록합니다.
- `isFavorite: true`는 초기 기본 즐겨찾기 시드입니다. 사용자가 별 버튼을 누른 뒤에는 브라우저 저장값이 우선합니다.
